Über uns


Warum können Programmierer nicht gut im Schach spielen? Weil sie immer die "Bugs" im Spiel suchen! Wenn ein Programmierer einen Kaffee bestellt, fragt er: "Haben Sie das auch in einer Version ohne Exceptions?" Und wenn er einen Fehler findet, sagt er: "Das ist kein Bug, das ist ein Feature!" Wenn Code nicht funktioniert, ist es wie ein Date: Man fragt sich, wo es schiefgelaufen ist. Und am Ende des Tages, wenn der Code läuft, fühlt man sich wie ein Jedi, der die Macht des Debugging gemeistert hat. Möge der Code mit dir sein!
